Design Notes
All screens are self-contained HTML files with embedded CSS and JavaScript. No external dependencies except Tailwind, Inter font, and Lucide icons.
Color system: Primary #2563EB, Navy #1E3C78, Success #16A34A, Warning #D97706, Error #DC2626, Portal #0EA5E9.
Typography: Inter font via Google Fonts. Consistent heading hierarchy and body text sizes across all screens.
Realistic data: All screens contain actual company names, job titles, and metrics. No placeholders or Lorem ipsum.
